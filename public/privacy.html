Privacy Policy

Wallet Impact App is a Progressive Web App (PWA) that helps you calculate financial impacts. This policy explains how we handle your data.

Data Storage

This app stores your input values locally on your device using localStorage. This means:

  • Your data never leaves your device
  • We cannot see your personal financial information
  • Data persists between browser sessions
  • You can clear this data anytime through your browser settings

Hosting & Infrastructure

The app is hosted on Firebase Hosting and served through Cloudflare's CDN. These services provide:

  • HTTPS encryption for all connections
  • Static file serving (no server-side processing)
  • Global content delivery

Google Analytics (Optional)

Analytics are only enabled if you explicitly set a Google Analytics ID.

If VITE_GA_ID is configured, the app will send anonymous usage events to Google Analytics:

  • mortgage_calc - When mortgage calculations are performed
  • student_calc - When student loan calculations are performed
  • gas_calc - When gas impact calculations are performed
  • grid_export - When CSV export is used
  • Copy and reset button usage

How to Disable Analytics

Option 1: Don't set VITE_GA_ID environment variable - analytics will be completely disabled

Option 2: Block Google Analytics in your browser using ad blockers or privacy extensions

Option 3: Use the version without GA4 integration

What We Don't Collect

  • Personal identification information
  • Financial account details
  • IP addresses or location data
  • Browsing history outside this app

Contact

This is an open-source project. For privacy concerns, you can:

  • Review the source code on GitHub
  • Run the app locally without analytics
  • Modify the code to remove any tracking
